[ubuntu入门手册]-4-安装搜狗拼音输入法

引言

Linux下优秀的拼音输入法有很多，但是能和Windows下的搜狗拼音输入法媲美的却很少，这篇文章实例演示安装搜狗输入法for linux官方版。

文章目录

  • 0×1.获取Ubuntu搜狗输入法官方deb安装包
  • 0×2.安装实例

0×1.获取Ubuntu搜狗输入法官方deb安装包

首先大家去搜狗拼音输入法for linux的官方网站，下载32位或64位安装包：搜狗拼音输入法for linux官方网站

打开终端，使用uname命令确认自己是32还是64位系统

						qing@qingsword.com.com:~$ uname -a
						...x86_64 x86_64 x86_64 GNU/Linux  //64位
						...i686 i686 i386 GNU/Linux       //32位

准备工作做完之后，下面直接安装。

0×2.安装实例

在Ubuntu14.04以上版本，安装步骤（本例在ubuntu16.04、18.04下测试安装成功）：

						#1.打开终端切换到deb安装包所在目录安装(大家安装的时候，deb文件的名称肯定与我这里不一样，可以使用ls查看下载到的文件名称，替换下面命令中的"sogoupinyin_2.2.0.0108_amd64.deb")
						qing@qingsword.com:~$ sudo dpkg -i sogoupinyin_2.2.0.0108_amd64.deb

						#2.解决软件依赖
						qing@qingsword.com:~$ sudo apt-get -f install

						#3.完成后重启计算机
						qing@qingsword.com:~$ sudo reboot

出于历史的原因，这里简单介绍一下Ubuntu12.04版本使用终端安装搜狗输入法的方法，此方法同样适用于12.04往后的高版本。

						#1.Ubuntu12.04版本中fcitx版本太老的，需要先更新它，12.04以上版本，跳过这一步
						qing@qingsword.com:~$ sudo add-apt-repository ppa:fcitx-team/nightly
						qing@qingsword.com:~$ sudo apt-get update
						#安装fcitx，会出现依存警告
						qing@qingsword.com:~/test$ sudo apt-get install fcitx
						#使用下面的命令来下载依存包
						qing@qingsword.com:~$ sudo apt-get -f install

						#2.使用dpkg命令安装deb包，这和双击deb安装的效果是一样的，sudo dpkg -i 后面是我们下载的deb包全名
						qing@qingsword.com:~$ sudo dpkg -i sogoupinyin_2.0.0.0072_amd64.deb

						#3.重启
