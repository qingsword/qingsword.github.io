[Linux实用命令]-1-超经典必会命令

引言

这篇文章玩转Linux必会的几个命令，包括：切换图形界面和命令行模式，显示文件夹类容，目录跳转，创建、删除、移动文件和文件夹，重启和关机等。

文章目录

  • 0×1.Linux图形界面和命令行界面的切换
  • 0×2.列出目录中的文件(ls)
  • 0×3.目录切换(cd)
  • 0×4.创建、移动、复制、删除文件的操作
  • 0×5.重启和关机

0×1.Linux图形界面和命令行界面的切换

进入Linux桌面环境后，可以使用键盘上的"Ctrl+Alt+F1~F6"组合键来切换不同的tty界面，Linux默认提供了6个命令行界面（F1-F6），比如"Ctrl+Alt+F1"就是切换到tty1：

					#最上面显示的是Ubuntu的版本计算机名称和tty号
					Ubuntu 16.04 LTS qingsword tty1

					#输入用户名和密码即可登录tty1
					qingsword login: qing
					Password:

在命令行模式下，想要切换回图形界面可以使用组合键"Ctrl+Alt+F7"；另外，如果不是从图形界面切换到tty模式，而是系统启动时候直接进入了命令行模式，在登陆后可以使用"startx"这条命令来启动图形界面（前提是图形界面有安装）。

Ps：在图形界面下快速启动命令行终端快捷键是"Ctrl+Alt+T"。

0×2.列出目录中的文件(ls)

在Linux命令行终端中使用的最多的命令是ls，它可以帮助我们列出当前目录下都有哪些文件，下面是一些实例：

					#使用pwd，查看当前是在哪个目录中，显示为"/home/qingsword"，这就是当前命令行所在目录
					qing@qingsword.com:~$ pwd
					/home/qingsword

					#切换到根目录
					qing@qingsword.com:~$ cd /

					#验证一下，确实在根目录下了
					qing@qingsword.com:/$ pwd
					/

					#查看根目录下面都有哪些文件和文件夹
					qing@qingsword.com:/$ ls
					bin   dev  home  lost+found  mnt  proc  sbin     srv  tmp  var
					boot  etc  lib   media       opt  root  selinux  sys  usr

					#可以使用两个ll详细输出，这个命令和ls -l效果相同
					qing@qingsword.com:/$ ll
					total 78
					dr-xr-xr-x.   2 root root  4096 Aug  9 01:21 bin
					dr-xr-xr-x.   5 root root  1024 Aug  8 23:54 boot
					drwxr-xr-x.  18 root root  3820 Aug  8 17:41 dev
					drwxr-xr-x.  88 root root  4096 Dec 24 10:39 etc
					...

					#添加参数a将显示隐藏文件和文件夹（同ls -al），在Linux中所有隐藏的文件和文件夹都是以符号.开头，例如下面的".autofsck"
					qing@qingsword.com:/$ ll -a
					total 86
					dr-xr-xr-x.  21 root root  4096 Aug  8 17:39 .
					dr-xr-xr-x.  21 root root  4096 Aug  8 17:39 ..
					-rw-r--r--.   1 root root     0 Aug  8 17:39 .autofsck
					dr-xr-xr-x.   2 root root  4096 Aug  9 01:21 bin
					...

上面为ls最常见的用法，下面是一些并不长用的方法：

					#使用递归模式，显示目录以及下级子目录，后面的more是当输出超出终端屏幕显示范围时，显示一个more,按任意键可以继续显示下面的内容（按ctrl+c可以提前结束命令显示内容的输出）。
					qing@qingsword.com:/$ ls -R | more

					#按文件修改时间先后顺序显示
					qing@qingsword.com:/$ ls -t

					#横排输出,中间用逗号分隔
					qing@qingsword.com:/$ ls -m

					#按大小排列，先大后小
					qing@qingsword.com:/$ ls -S

除此之外ls会显示不同颜色，下面是这些颜色的含义：

蓝色—目录
绿色—可执行文件
红色—压缩文件
浅蓝色—链接文件
粉红色或暗紫色—图片文件
黄色—设备文件
灰色—其他文件

可以切换到/dev目录下用ls查看一下，会看到大部分为设备文件。

0×3.目录切换(cd)

使用cd（Change Directory）命令可以在不同目录之间切换，下面是一些实例：

					#当前目录是根目录
					qing@qingsword.com:/$ pwd
					/

					#使用cd切换到/dev这个目录
					qing@qingsword.com:/$ cd /dev
					qing@qingsword.com:/dev$ pwd
					/dev

					#这条命令可以用来回到前一个工作目录，即根目录
					qing@qingsword.com:/dev$ cd -
					/

					#进入当前用户的家目录（在Linux终端中，～代表当前用户的家目录）
					qing@qingsword.com:/$ cd ~
					qing@qingsword.com:~$ pwd
					/home/qingsword

					#返回上一层目录（一个点代表当前目录，两个点代表当前目录上一层目录）
					qing@qingsword.com:~$ cd ..
					qing@qingsword.com:/home$ pwd
					/home

0×4.创建、移动、复制、删除文件的操作

创建目录，或者创建嵌套目录：

					#在当前用户的家目录中，创建一个单层目录
					qing@qingsword.com:~$ mkdir qing
					qing@qingsword.com:~$ ls
					qing

					#试着创建一个嵌套的目录，提示错误
					qing@qingsword.com:~$ mkdir top/sub/sub2
					mkdir: cannot create directory `top/sub/sub2': No such file or directory

					#添加-p参数才能创建多层的嵌套目录
					qing@qingsword.com:~$ mkdir -p top/sub/sub2

删除目录或者文件：

					#强制删除刚才创建的top目录"rm -rf"后面可以接任何目录或者文件（删除目录或文件的命令，只要会这一条就足够了）
					qing@qingsword.com:~$ rm -rf top

创建文本文件：

					#使用touch命令可以创建一个空文本文件
					qing@qingsword.com:~$ touch testfile

					#使用echo命令将hello这几个字符写入这个文件中
					qing@qingsword.com:~$ echo hello>>testfile

					#使用more命令显示这个文件中的内容
					qing@qingsword.com:~$ more testfile
					hello

					#追加一行，再次显示
					qing@qingsword.com:~$ echo My name is QingSword>>testfile
					qing@qingsword.com:~$ more testfile
					hello
					My name is QingSword

					#创建文本的方式不止这一种，还有下一篇文章会介绍到的vim以及图形化的文本编辑器gedit等等，touch是创建空文本最方便的一种方式

复制文件或文件夹：

					#首先创建两个文件夹
					qing@qingsword.com:~$ mkdir dir1
					qing@qingsword.com:~$ mkdir dir2
					qing@qingsword.com:~$ ls
					dir1  dir2  qing  testfile

					#将testfile这个文本文件复制到dir2目录中
					qing@qingsword.com:~$ cp testfile dir2
					qing@qingsword.com:~$ ls dir2
					testfile

					#将dir2目录复制到dir1目录中,复制非空目录要添加参数-r
					qing@qingsword.com:~$ cp -r dir2 dir1

					#复制多个目标到一个文件夹中，将dir1和dir2这两个文件夹以及testfile这个文本文件，复制到后面的qing文件夹中。
					qing@qingsword.com:~$ cp -r dir1 dir2 testfile qing

移动目录或文件(剪切)：

					#vm可以操作目录和文件，和cp一样可以一次性操作多个目标
					#将dir1移动到qing目录中
					qing@qingsword.com:~$ mv dir1 qing
					qing@qingsword.com:~$ ls
					dir2  qing  testfile

					#将testfile dir2一起移动到qing目录中
					qing@qingsword.com:~$ mv testfile dir2 qing
					qing@qingsword.com:~$ ls
					qing

0×5.重启和关机

使用下面的命令重启和关机(这个命令必须使用sudo或使用root权限才能执行)，sudo能让普通用户暂时获得root的执行权利：

					#一分钟后关机
					qing@qingsword.com:~$ sudo shutdown -h 1

					#如果是定时关机，可以打开另外一个终端，使用下面这个命令取消关机
					qing@qingsword.com:~$ sudo shutdown -c

					#马上关机
					qing@qingsword.com:~$ sudo shutdown -h now

					#下面这两条命令也是马上关机
					qing@qingsword.com:~$ sudo init 0
					qing@qingsword.com:~$ sudo poweroff

					#下面这两条命令都能马上重启
					qing@qingsword.com:~$ sudo reboot
					qing@qingsword.com:~$ sudo init 6

除了定时关机外，最常用的两个命令就是reboot（重启）和poweroff（关机）。
