[mysql基础文档]-1-搭建MySql学习环境

引言

这一系列的文章将给大家带来SQL相关的语法知识，讲解最常用的SQL语句，用实例的形式描述其功能，方便大家的学习与查阅。

本文介绍目前最常用的几款数据库，如何在Ubuntu系统中安装Mysql，最后介绍MySQL官方推荐的数据库图形管理软件WorkBench在ubuntu下的安装与使用。

本系列文章主要使用MySql数据库，系统环境为UbuntuServer16.04，虽然系统环境为Linux，但Windows环境的朋友完全不需要担心，因为本系列的所有sql命令都能在windows环境下运行，与linux不同的部分，会在文中注明。

0×1.什么是数据库

数据库是——存储电子文件的处所，用户可以对文件中的数据进行新增、截取、更新、删除等操作。 它能为多个用户共享、具有尽可能小的冗余度的特点、是与应用程序彼此独立的数据集合（百科）。

简而言之，数据库是以某种约定的数据结构高效的储存数据，并在用户需要时通过检索展现给用户。

如今较为常见的数据库有Oracle、MySQL、IBM DB2、Microsoft Access、Microsoft SQL Server。本系列文章中所使用的大部分SQL语句均能够在这五种数据库中执行，但他们之间仍然有一小部分不同之处。

0×2.Ubuntu如何安装MySql

1）安装，确保网络畅通打开终端，使用apt-get在线安装：

					/**
					 -- sudo apt-get update 更新系统源列表
					 -- sudo apt-get install mysql-server 安装mysql服务端
					 -- 安装过程中会让我们设置root的密码，这个密码很重要，登陆数据库时候要用到
					 **/
					www@qingsword.com:~$ sudo apt-get update && sudo apt-get install mysql-server

					--workbench为mysql官方发布的数据库可视化管理软件，可图形化管理数据库（类似phpmyadmin）
					www@qingsword.com:~$ sudo apt-get install mysql-workbench

2）连接到数据库

					--在本地，使用root用户登陆数据库
					www@qingsword.com:~$ mysql -u root -p
					Enter password: <--在这里输入刚才安装时候设置的root密码

					--登陆成功提示符如下，输入exit可以退出
					mysql>

0×3.Windows如何安装MySql

1）下载

建议到[Mysql官方网站]下载，如果是学习用，下载社区版就足够了（MySQL Community Edition (GPL)），企业版功能很全面（MySQL Enterprise Edition (commercial)），但安装文件相对大很多；

版本选择注意事项：

分为压缩版（ZIP）和安装版（MSI），压缩版需要手动更改配置文件使用命令行安装，建议大家选择MSI安装版；其次，根据自己系统架构，选择32或64位安装包。下载相关图文教程请参考——[如何在官网上下载可安装版的MySQL数据库]（最新的mysql页面能自动检测系统架构，所以文中第五步可能和你看到的不一样，只会显示一个MSI下载地址和两个ZIP地址，选择MSI下载即可）

2）安装

相关图文教程请参考——[MySQL5.7安装图文教程]

3）连接到数据库

开始菜单MySQL文件夹下MySQL Server中提供了一个快速连接接口"MySQL Command Line Client”，这个就是数据库的命令行管理接口了，打开它会提示： 

					--输入密码连接到数据库，安装时候设定的超级用户密码，即可连接到数据库
					Enter password: **********
					Welcome to the MySQL monitor. 
					.....
					mysql>
